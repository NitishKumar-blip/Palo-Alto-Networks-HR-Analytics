# Palo-Alto-Networks-HR-Analytics
Creating of dashboard using given csv file for  data driven decision for the company.

Examining the Cause of Attrition in the Company.

OVERVIEW :

The HR Analytics Dashboard is a data visualization and analysis project designed to help organizations understand workforce trends, employee attrition, job satisfaction, performance, and other key HR metrics. The dashboard provides interactive visualizations and actionable insights for HR professionals and management.

FEATURES :

1.Employee Attrition Analysis
2.Department-wise Employee Distribution
3.Gender Diversity Analysis
4.Job Satisfaction Tracking
5.Age Group Analysis
6.Salary and Compensation Insights
7.Performance Evaluation Metrics
8.Interactive Filters and Visualizations


TECHNOLOGY USED :

1.Python 
2.Pandas
3.NumPy 
4.Matplotlib 
5.Seaborn 
6.Tableau 
7.Power Bi

The dataset contains employee-related information such as:

1.Employee ID
2.Age
3.Gender
4.Department
5.Job Role
6.Monthly Income
7.Years at Company
8.Job Satisfaction
9.Attrition Status

KEY INSIGHTS :

1.Identify factors contributing to employee attrition.
2.Analyze employee satisfaction levels.
3.Compare department performance.
4.Understand workforce demographics.
5.Support data-driven HR decision-making.

FUTURE ENHANCEMENTS

1.Predictive Attrition Modeling
2.Employee Performance Forecasting
3.Real-time Data Integration
4.Advanced HR KPI Monitoring
